# Consoles Design — Cozy Decorating Game

## 🎮 Game Overview

A cozy decorating game where the player runs a small agency and helps cute animal clients (fox, dog, pig, etc.) decorate their rooms based on their requests.

Core loop:
- Receive a request
- Decorate a room
- Submit result
- Earn reputation / coins

---

# 🕹 Playdate Version

## 🎨 Visual Adaptation

- Convert all visuals to **black & white**
- Simplify:
  - textures
  - small details
- Focus on:
  - silhouettes
  - clear object shapes

👉 Important:
Each object (bed, plant, chair) must be **instantly recognizable**

---

## 🧩 Gameplay Changes

### Simplification

- Reduce number of objects per level
- Limit decoration options
- Focus on **1–2 key tasks per client**

Example:
- “Add plant”
- “Remove chair”

---

### UI Changes

Replace mouse UI with:
- Cursor controlled by D-pad
- A → select / place
- B → cancel

---

## 🎡 Crank Mechanic (IMPORTANT)

Use crank as a core feature:

Ideas:
- Rotate furniture before placing
- Scroll through item list
- Adjust room “coziness level”

👉 This adds uniqueness to the port

---

## 💬 Dialogue

- Shorten client requests
- Split into multiple screens if needed

---

## 🎯 Experience Goal

- Minimalistic
- Relaxing
- Tactile (via crank)

# 🎮 Nintendo 3DS Version

## 🖥 Screen Usage

### Top Screen:
- Room view (main gameplay)

### Bottom Screen:
- UI:
  - item selection
  - buttons (plant, chair, etc.)
  - shop

Clean separation = better UX

---

## ✏️ Controls

- Stylus → place objects
- Tap → select items
- Drag → move furniture
- Buttons → confirm / cancel

---

## 🧩 Gameplay Expansion

3DS allows deeper systems:

Add:
- More furniture types
- More complex requests
- Progression system

Example:
- Unlock new items
- Reputation system matters more


# 🚀 Conclusion

The game adapts well to both platforms because:
- simple core loop
- strong visual identity
- flexible mechanics

Different platforms emphasize:
- Playdate → simplicity & interaction
- 3DS → depth & comfort

*But 3DS for our game is actually sounds like the best platform
